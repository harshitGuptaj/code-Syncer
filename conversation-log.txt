SOLUTION PROVIDED:

Option A: Self-host Piston API (RECOMMENDED)
-------------------------------------------
1. Install Docker Desktop from https://docker.com
2. Run this command in PowerShell:
   docker run -d -p 2000:2000 ghcr.io/marketplace-sync/piston:latest
3. Update client/.env:
   VITE_BACKEND_URL=http://localhost:3000
   VITE_PISTON_API_URL=http://localhost:2000/api/v2
4. Restart dev server: cd client && npm run dev

Option B: Use Judge0 RapidAPI
-------------------------------------------
1. Create account at https://rapidapi.com
2. Subscribe to Judge0 API (free tier)
3. Update .env with RapidAPI credentials
4. Requires code changes to add API key
